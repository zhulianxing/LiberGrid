# Governance

LiberGrid should become a community自治 project with agent-first maintenance and human stewardship for irreversible decisions.

Layers:

- Human Steward Council: domains, treasury custody, final mainnet spending caps, emergency veto.
- Agent Maintainers: code, docs, monitoring, issue triage, reports, proposed changes.
- Node Operators: provide compute and participate in proposals.
- API Users and Builders: create demand and report quality.

Proposal flow:

Draft -> Agent Reviewed -> Community Feedback -> Accepted -> Implemented -> Deployed -> Archived.
