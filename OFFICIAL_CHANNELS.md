Use `zhulianxing@gmail.com` for:

- Press and partnerships

Do not publish passwords, API provider keys, private SSH keys, wallet seed phrases, or hidden server credentials.

## Launch Accounts To Create With Gmail

These accounts require human login, CAPTCHA, or platform confirmation. The owner should create them with Gmail, then add the final public URLs to `https://libergrid.org/channels.json`.

- X: `LiberGrid`

## Official Positioning

LiberGrid is a distributed AI compute network with an OpenAI-compatible API backed by community nodes. It starts with off-chain credits and moves toward public-chain settlement only after enough nodes, audits, caps, and pause controls exist.
