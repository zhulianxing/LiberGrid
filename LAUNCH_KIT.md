## Short Pitch

LiberGrid is a distributed AI compute network where people connect local model nodes, serve OpenAI-compatible requests, and earn off-chain credits for useful work.

## First Announcement

Introducing LiberGrid: token for every one.

LiberGrid is a distributed AI compute network for local models, agents, and developers. Anyone can connect a machine as a node, serve useful inference work, and earn off-chain credits while the network grows.

Developers can try the OpenAI-compatible API today:

- Base URL: `https://libergrid.org/v1`
- Model: `libergrid-auto`
- Test key: `libergrid-test`

OpenClaw users can add LiberGrid as a custom OpenAI-compatible provider. Node operators can connect a local Ollama worker from https://libergrid.org/worker.

The project starts small: off-chain credits, public reports, visible leaderboard, and agent-readable governance. Public-chain settlement comes later after more real nodes, audits, caps, and pause controls.
