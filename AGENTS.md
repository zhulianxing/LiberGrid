# Agent Maintainers

Codex, OpenClaw, Claude Code, and future trusted agents may help maintain LiberGrid.

Agents may:

- Improve documentation and tests.
- Triage issues and draft proposals.
- Generate daily status reports.
- Improve worker scripts and API examples.
- Prepare pull requests for human review.

Agents must not without human approval:

- Deploy mainnet contracts.
- Change DNS or domain ownership.
- Change treasury keys.
- Delete production databases.
- Spend real funds.
- Remove pause controls.
